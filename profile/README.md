USDA's livestock disaster programs — led by the **Livestock Forage Disaster Program (LFP)**, which has paid producers more than $12 billion since 2008 — hinge on two inputs: county **Normal Grazing Periods** and the weekly **US Drought Monitor**. This project rebuilds the science behind those inputs as reproducible, publicly auditable workflows, and archives the authoritative data that federal drought relief runs on.

Every archive here is versioned, openly licensed, documented, and served at [sustainable-fsa.com](https://sustainable-fsa.com).
